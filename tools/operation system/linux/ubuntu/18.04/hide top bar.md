# hide top bar

调节顶部状态栏

## 安装

```shell
sudo apt-get install gnome-shell-extension-autohidetopbar
```

按下快捷键 `Alt + F2`，输入 `r`， 然后按下 `Enter` 回车键。目的是重启 `gnome-shell`

在 `dash` 中搜索 `tweak` ，点击打开
