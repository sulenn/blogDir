# 远程服务器连接

- FileZilla：图形化界面操作，非命令行形式
    需要手动设置桌面快捷方式

    ![2](http://ww1.sinaimg.cn/large/006alGmrly1g1njhv7hhkj30x50qc41o.jpg)

- putty：可以进行命令行操作
    添加主机IP，默认端口为 22
    保存 Session 为 qiuing ，下次就可以点击 qiubing 进行远程连接，不需要再次输入 IP address
